2D-Platformer on Android

This project was my first attempt to work on mobile app development. The program runs on android devices, and I also included a desktop Java version with a level editor. Working on this game turned out to be very challenging, as I wanted to avoid using any external libraries to handle mechanics in order to learn. The main features include tile-based graphics, imageHandler class, program auto reshapes to fit any mobile screen size, a text-to-graphics level editor which contains 3 layers of information per tile, a collision detection mechanism, manual sprite animation, responsive screen scrolling, object oriented entities, mini physics engine and multi-thread processing for loading the content. The main engine of this project is done, as with the text base level editor, additional content can be implemented without the need of additional code.

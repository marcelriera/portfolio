Snake Minigame in Javascript

This simple recreation of the classic Snake game was made with the purpose of getting to know Javascript language. Developing this program was useful to see the similarities and differences between Java and Javascript common syntaxes. Even though unneeded, the program implements a basic object oriented approach, mainly to get used with javascript object syntaxes.

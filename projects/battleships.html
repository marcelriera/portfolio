Battleships Classic Game in Java Swing

This project was my first real attempt to develop software using Swing GUI arquitecture. The challenge with this project was to maximize the use of Swing elements in the implementation of the game. The end result contains six classes, most of them being Swing UI extended elements modified to suit the program. Highlighted features of this project include the use of a JButton grid for the game board, a GamePanel class extension for both the player an oponent, the development of a static ImageHandler class to resize, rotate and crop images, and a basic A.I. to play with the computer.

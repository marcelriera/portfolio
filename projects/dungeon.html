Card Maze in Java Swing

The purpose of this project was specifically to work on an app that benefited from the use of object oriented programming. Card Maze is a card base game which introduces the player in a 3x3 board of cards (including himself) and allows him to interact vertically and horizontally with the board. This application contains a total of 23 classes, which are classified in GUI, Main and Card classes. The application relies heavely on object structures, as each different card is an object, containing all his specific functionality from within, and extending a parent Card abstract class. By implementing the software with this approach the game engine benefits of using polymorphism to handle different cards, hence making the implementation of aditional content easy as they are imediatelly supported.
